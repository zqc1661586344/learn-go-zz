# 回溯
## 代码模板
```go
func backtracking(参数) {
    if 终止条件 {
        存放结果
        return
    }

    for (选择：本层集合中元素（树中节点子的数量就是集合的大小）) {
        处理节点
        backtracking(路径，选择列表) // 递归
        回溯，撤销处理结果
    }
}
```
